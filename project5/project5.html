publishing

  • header nav 에 position: fixed를 사용하여 웹사이트의 상단에 항상 위치하도록 배치
  • header nav 에 justify-content: space-between을 사용하여 로고, 메뉴, 로그인의 간격 조정
  • display: flex가 사용되는 요소에 class명 flex를 사용하여 필요한 요소에 flex가 적용되도록 작성
  • header main text 에 position: absolute를 사용하여 메인 이미지의 가운데 부분으로 배치
  • 정사각형 모양의 요소에 aspect-ratio: 1 을 사용하여 완벽한 정사각형 모양으로 디자인
  • 변수 var 를 이용하여 보다 정확하고 신속한 코드 작성
  • flex-direction: column, flex-direction: row 를 이용하여 요소의 flex 방향 설정
  • justify-content: center, align-items: center 를 사용하여 텍스트 박스 안의 텍스트가 가운데로 위치하도록 설정
  • youtube 에서 퍼가기를 이용하여, html 파일에 삽입 autoplay=1&mute=1&loop=1 값 입력 자동 재생 되도록 설정
  • 카카오맵 지도는 지도를 표시할 div를 생성후, sdk.js?appkey=(발급받은 APP KEY) 를 입력하여 카카오맵 지도 삽입

solution

리뉴얼 전 사이트의 문제점 파악

  • index 페이지에 회사의 설명이나 보여지는 부분이 부족한 점
  • 깔끔한 느낌을 갖고 있지만 더욱 세련된 스타일을 필요로 함

리뉴얼 전 사이트의 문제점 해결

  • index 페이지에 회사소개 영상과, 제품설명, 제품 사용 이미지 추가
  • 격자 형식의 이미지를 삽입하여 세련되고 정돈된 느낌을 강조

리뉴얼을 진행하며 생긴 문제점

사이트의 유튜브 하단에 메인 이미지가 모여있는 섹션에서 이미지가 많이 들어가다 보니, 계획한 디자인과 다르게 서로 어긋나게 배치가 됨.

유튜브 영상을 추가 후 autoplay=1 값을 입력하였지만 자동재생이 안되 는 문제가 발생하였다.

리뉴얼을 진행하며 생긴 문제점 해결

  • 이미지가 어긋나게 배치된 문제점 해결
  • img {width:750px;} -> img {width: 50%;}
  • img 의 width 값을 px이 아닌 50% 값을 입력 후 해결
  • 유튜브 영상 자동재생 문제점 해결
  • <src="controls=0&autoplay=1&loop=1"> ↓ <src="controls=0&autoplay=1&mute=1&loop=1">
  • &mute=1 값을 추가 후 해결
