기술서 보러가기

프로젝트 결과

피그마 웹페이지 디자인과 거의 흡사하게 제작되었으며, 깔끔하고 서정적인 느낌에 맞추어 java-script 동작까지 구현하게 되었다. 페이지를 제작하면서 수업 시간에 배우게 된 plug in을 이용하여 움직임이 다채로운 웹페이지를 완성하였다.

프로젝트 평가

    부족한점
  • Git을 통한 팀원들과 웹페이지 제작을 처음 해본결과 git의 정보 충돌이 많이 일어나게 되었다. 하지만 구글링, 팀원들과 소통을 하며 점차 문제를 해결해 나갔으며, git의 활용도를 정말 많이 높여준 팀프로젝트였다.
    잘한 점
  • 처음에 나오는 메인 이미지 슬라이드가 밑부분이 잘려서 동작하였는데, 팀원들과 의견을 나누 면서 같이 작업 후에 오류를 해결할 수 있었다.
  • 메인 이미지 다음으로 사진이 크게 들어가는 제품 소개 이미지가 워낙 크게 들어가서, 이미지 가 깨지는 현상이 발생하였다. 팀원들과 끝까지 포기하지 않고 포토샵 툴을 이용하여 문 제를 해결하였다

개인리뷰

학원 수업을 진행하며 git을 개인적인 작업 용도로만 사용하다가 직접 팀원들과 교류하며 git 을 써보니 처음엔 굉장히 어렵고 어색한 느낌을 받았다. 하지만 작업을 진행하다 보니 적응도 되고, git의 중요성을 확실하게 알게 된 팀 프로젝트였다.

자바스크립트, 플러그인을 배운 후 처음 적용해 보는 작업이라 처음엔 조금 막막하고 잘할 수 있을까 고민이 되었지만 팀원들과 서로 도우며 작업해 보니 의외로 작업이 그리 어렵지 않다는 느낌을 받을 수 있었다. 조금 더 어려운 작업도 가능할 것 같다는 생각이 들어서 조금 아쉽다는 생각도 들었다
